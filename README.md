# BondKeeper

## Project Description
BondKeeper is a responsive web application designed to help users maintain meaningful relationships by tracking interactions with friends. It allows users to monitor when they last contacted someone, set communication goals, and log interactions like calls, texts, and video chats in a clean and intuitive interface.

---

## Technologies Used

- Next.js - React framework for building the application  
- App Router (Next.js) - Routing and navigation  
- Tailwind CSS - Styling and responsive design  
- Recharts - Data visualization (analytics charts)  
- React Toastify (or similar) - Toast notifications  
- JSON - Local data storage for friend profiles 

---

## Key Features

### Friend Management
- View all friends in a structured card layout  
- Each friend displays contact status, tags, and last interaction  
- Click a card to view detailed profile  

### Quick Check-In
- Log interactions (Call, Text, Video) instantly  
- Automatically updates the Timeline  
- Displays toast notifications on actions  

### Timeline Tracking
- View complete interaction history  
- Includes date, type (icon), and title  
- Filter interactions by Call, Text, or Video  

---

## Additional Features

- Friendship Analytics page with Pie Chart using Recharts  
- Fully responsive design (mobile, tablet, desktop)  
- Loading animation while fetching data  
- Custom 404 page for invalid routes
- Highlighted active navigation links  
- Styled footer section

## Installation and Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/asm-ayesha/bond-keeper.git
   ```

2. Navigate to the project folder:
   ```bash
   cd bond-keeper
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

5. Open in browser:
   ```
   https://bond-keeper.vercel.app/stats
   ```
